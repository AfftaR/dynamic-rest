insert(parts, leaf_value, update=False)[source]¶

Add a list of nodes into the tree.

The list will be converted into a TreeMap (chain) and then merged with the current TreeMap.

For example, this would insert [‘a’,’b’,’c’] as {‘a’:{‘b’:{‘c’:{}}}} into an empty TreeMap.

Parameters:
  • parts – list of nodes representing a chain
  • leaf_value – value to insert into the leaf of the chain
  • update – whether or not to update the leaf with the given value or to replace the value. Default: False (replace)

get_ordering(request, queryset, view)[source]¶

Return an ordering for a given request.

DRF expects a comma separated list, while DREST expects an array. This method overwrites the DRF default so it can parse the array.

get_valid_fields(queryset, view)[source]¶

Return valid fields for ordering.

Overwrites DRF’s get_valid_fields method so that valid_fields returns serializer fields, not model fields.

remove_invalid_fields(queryset, fields, view)[source]¶

Remove invalid fields from an ordering.

Overwrites the DRF default remove_invalid_fields method to return both the valid orderings and any invalid orderings.
